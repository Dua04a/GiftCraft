# GiftCraft

## Overview

GiftCraft is a C# console application that allows users to create a personalized gift box based on the selected occasion and budget.

The project was created to practice Object-Oriented Programming (OOP) concepts and apply design patterns in a simple gift customization system.

---

## Features

- Select an occasion
- Set a budget
- Create a personalized gift box
- View the final gift summary
- Calculate the total cost

---

## Design Patterns

- **Builder Pattern** – Builds customized gift boxes step by step.
- **Factory Pattern** – Creates gift objects based on the user's selections.
- **Strategy Pattern** – Applies different gift selection strategies based on the selected occasion.

---

## Technologies Used

- C#
- .NET
- Object-Oriented Programming (OOP)

3. Run the application.

```bash

```
